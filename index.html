L Omni-Language

v2.0 · Self-Hosting · AI-Native
Meta-Language Autograd + Tensor Multi-Transpiler C / Python / JS Runtimes x86_64 Assembly Turing Tape AI Training IR

"Universal Meta-Language: from Turing Tape to AI Gradients."

A fully self-hosting meta-language (L-AST) that bridges low-level architectures (x86_64 Assembly, Turing Machine Tape) with native AI mathematics (Autograd, Tensors, SGD). Symmetric C / Python / JS interpreters. Multi-target transpiler compiles to 6 outputs. Highlights: self_host.l (L parser written in L itself), ai_classifier.l (50-epoch SGD, Loss → 0 in 25 steps, verified on all 3 runtimes).

"튜링 테이프부터 AI 그래디언트까지 — 범용 메타 언어."

저수준 아키텍처(x86_64 어셈블리, 튜링 머신 테이프)와 고수준 AI 연산(오토그래드, 텐서, SGD)을 잇는 완전 자기 호스팅 메타 언어입니다. C / Python / JS 인터프리터 대칭 구현, 6개 타겟 멀티 트랜스파일러. 하이라이트: self_host.l (L 언어로 쓴 L 파서), ai_classifier.l (50회 SGD, 25회에 Loss 0 수렴, 3개 런타임 모두 검증 완료).
